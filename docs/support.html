Help with Is My Post Removed? for Reddit

Frequently Asked Questions

The extension isn't showing any badges on my posts

Make sure you're viewing your own posts while logged in to Reddit. PostGhost only checks posts authored by the currently logged-in user. Try refreshing the page — badges appear after a brief check.

I see a badge but I think it's wrong

PostGhost reads Reddit's own removed_by_category field. If you believe a result is incorrect, try opening your post in an incognito/private window (without logging in). If you can't see the post there, it's been removed — PostGhost is correct. If you can see it, please report the false positive.

Does this work on new Reddit and old Reddit?

Yes. PostGhost works on both reddit.com (new design) and old.reddit.com. The underlying .json endpoint is the same.

Will Reddit ban me for using this?

No. PostGhost makes a single standard HTTP request to Reddit's public .json endpoint per post — the same request your browser makes when loading a page. It does not scrape, does not use unofficial APIs, and does not violate Reddit's terms of service.

Does PostGhost work on mobile?

No. PostGhost is a Chrome desktop extension. Chrome on Android and iOS does not support extensions. For mobile, you can manually check by opening your post URL in a private/incognito tab.

Can I check other people's posts?

No. Reddit only exposes the removed_by_category field to the post author. PostGhost can only check posts you authored while you're logged in.

Troubleshooting

Extension stopped working after a Reddit update

Reddit occasionally changes their page structure. PostGhost reads the .json endpoint (which is more stable than the HTML), but if something breaks:

  1. Check for extension updates in chrome://extensions
  2. Try disabling and re-enabling the extension
  3. If the issue persists, report it on GitHub

Badges appear slowly

PostGhost waits 30-60 seconds after detecting a new post before checking, because Reddit's filter pipeline takes time. For existing posts on your profile page, badges should appear within 1-2 seconds of page load.
